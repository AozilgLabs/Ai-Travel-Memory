# AI Travel Memories

## これ何？

旅行が終わると、
大量の写真を撮る。

でも、

Google Photosに保存して終わる。

友達にも送らない。

SNSにも投稿しない。

写真は眠ったまま。

この「旅行後の面倒」をAIで全部なくしたい。

写真を入れるだけで、

旅行が完成する。

それを作るプロジェクトです。

---

## MVP

- 写真アップロード
- AIがベスト写真を選ぶ
- リール動画生成
- ダウンロード

---

## ゴール

旅行から帰ってきたら、

写真を放り込むだけ。

数分後には

友達へ送りたくなる動画が完成している。
