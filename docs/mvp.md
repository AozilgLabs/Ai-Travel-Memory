# AI旅行思い出プラットフォーム - MVP仕様書 v0.1

## プロジェクト概要

### コンセプト

**写真を入れるだけ。旅行が完成する。**

旅行中に撮影した大量の写真をアップロードするだけで、AIが自動的に整理・選定・動画生成を行い、「共有したくなる思い出」を完成させるサービス。

---

## 背景

現在の旅行写真は、

- Google Photosへ保存する
- 見返さない
- 友人へ共有しない
- SNSへ投稿しない

というケースが非常に多い。

原因は、

「写真を撮った後の整理・選定・共有」が面倒だから。

本サービスでは、その工程をAIが代行する。

---

# MVPの目的

MVPでは、

**「写真をアップロードするだけで、共有できる思い出が完成する」**

という体験を提供する。

Google Photosの代替ではなく、

Google Photosに保存する前後で利用するサービスを目指す。

---

# MVPの対象ユーザー

旅行・イベントで大量の写真を撮影する人

例

- 旅行
- デート
- 水族館
- テーマパーク
- 花火大会
- カフェ巡り
- ドライブ

※MVPでは旅行に限定せず、「1回のお出かけ」を対象とする。

---

# MVP機能一覧

## 1. 写真アップロード

### 必須

- 複数写真を一括アップロード
- ドラッグ＆ドロップ対応
- フォルダごとアップロード対応

### 対応形式

- JPEG
- JPG
- HEIC

PNG・RAWは後日対応予定

---

## 2. AI写真解析

アップロードされた写真を解析する。

解析内容

- 撮影日時
- GPS情報
- 重複写真検出
- 類似写真検出
- ブレ検出
- 明るさ
- ピント
- 構図

解析結果からベスト写真を自動選定する。

---

## 3. ベスト写真生成

例

入力

```
2,381枚
```

↓

出力

```
48枚
```

AIが旅行を代表する写真を選定する。

---

## 4. AIリール動画生成

AIが選定した写真から動画を作成する。

MVPでは

- 15秒
- 30秒

の2種類を生成。

演出はシンプルなスライドショーでよい。

高度な動画編集はVersion2以降。

---

## 5. 完成画面

解析終了後、

旅行（イベント）の成果物を表示する。

例

```
北海道旅行

写真
2,381枚

ベスト写真
48枚

動画
30秒
```

ここがサービスのメイン画面になる。

---

## 6. ダウンロード

ダウンロードできるもの

- リール動画(mp4)
- ベスト写真(zip)

# UXで最も重要なこと

Google Photosとの違いは、

である。

```
写真をアップロードする

↓

待つ

↓

完成
```

だけで良い。

↓

↓

この体験を5分以内で実現できること。
